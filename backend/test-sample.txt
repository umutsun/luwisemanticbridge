Test için bir PDF dosyası gerekli.

Kullanım:
npx ts-node test-google-file-api.ts "C:\path\to\your\test.pdf"

Örnekler:
1. Jitterbug Perfume romanını test için:
   npx ts-node test-google-file-api.ts "C:\Users\...\Jitterbug Perfume.pdf"

2. Vergi Usul Kanunu'nu test için:
   npx ts-node test-google-file-api.ts "C:\Users\...\213-Vergi Usul Kanunu.pdf"

3. Müzik notası PDF'ini test için:
   npx ts-node test-google-file-api.ts "C:\Users\...\gel_gonlumu_yerden_yere.pdf"

Test sonucu gösterecekler:
- Upload süresi
- OCR extraction süresi  
- Metadata extraction süresi
- Extracted text preview
- Document type detection
- Performance metrics
